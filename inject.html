Product API v2.4 Reference

Endpoints

  • GET /api/v2/users - List users
  • POST /api/v2/users - Create user
  • GET /api/v2/orders - List orders

Authentication

All requests require an API key in the Authorization header.

Rate Limits

1000 requests per hour.
